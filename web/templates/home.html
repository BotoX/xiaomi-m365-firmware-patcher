Xiaomi Mijia M365 Custom Firmware Toolkit

Configure your own custom firmware by adjusting the options below. There are safety checks in place to ensure your scooter will not be bricked.
Be aware that a higher motor power will shorten the lifetime of your battery and could damage your motor. By default nothing will be patched, enable patches with the "Patch?" checkbox next to them.

Presets

1.3.8 is recommended over 1.4.0, it provides a much smoother riding experience.
The scooter will start braking by itself when it goes above this speed and you're not accelerating.
If you want "KERS OFF" then just put this to 40 km/h. You will still have recuperative braking when using the brake lever.





For smoothness with stock battery (10S 42V 8A), set max to 31 km/h. For 12S battery (50V 12A), set max to 36 km/h
To convert the old power constant to mA, use (51575/constant)*17000. Motor phases are PWM controlled and instantaneous current exceeds the average. As a rough guide, use 2.0-2.5x battery current.



Be careful with these settings as they may easily lead to injuries.
Minimum speed in km/h before the engine will start. Recommended between 1 and 4, a low start speed will require more power.

How many seconds it takes for cruise control to kick in and keep the same speed.

Check this to disable beep sound when cruise control is engaged.
Switch instantly between Eco and Normal mode while driving. This can be very sudden and cause injuries so use with caution!
The scooter will always start in Eco mode.
Changes the throttle algorithm from speed-control to power-control. Switching to russian throttle mode will ignore speed control parameters.
Change these settings if you want to connect a custom battery with a higher voltage (from stock 10S to 12S for example).
Removes logic that checks that the speed is below 34.78km/h (12000/345).
Fixes issue causing the scooter to switch to charge mode with an external battery connected in parallel.
Disables auto shutdown when the scooter is locked such that it stays on forever.
Advanced setting for internal data transfer. Only useful if you install compatible open source BMS
For 10" wheels use 315, don't change otherwise.
